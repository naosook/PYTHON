# Atualizando Variáveis

Quando você atribui um novo valor a uma variável já existente, você está atualizando essa variável. Em Python, isso é feito simplesmente atribuindo um novo valor ao nome da variável. Por exemplo: 

```python
idade = 25 
idade = 30  # Atualizando o valor da variável idade
```

Neste exemplo, a variável `idade` é inicialmente definida como 25 e depois é atualizada para 30.

# Expressões

Expressões em Python são combinações de variáveis, valores e operadores que produzem um novo valor. Elas podem ser tão simples quanto uma única variável ou tão complexas quanto uma série de operações matemáticas. Por exemplo:

```python
idade = 25
idade_em_5_anos = idade + 5  # Expressão que calcula a idade daqui a 5 anos
```

Neste exemplo, a expressão `idade + 5` adiciona 5 à variável `idade`, produzindo o valor `30`, que é então atribuído à variável `idade_em_5_anos`.

# Números

Python suporta diferentes tipos de números, incluindo:

- Inteiros (por exemplo, `10`)
- Floats (por exemplo, `3.14`)
- Complexos (por exemplo, `2 + 3j`)

Você pode usar esses tipos de números em operações matemáticas e cálculos. Por exemplo:

```
inteiro = 10
flutuante = 3.14
complexo = 2 + 3j
```
